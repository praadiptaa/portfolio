
# Portofolio Pribadi - Next.js Monochrome Minimalis

Website portofolio pribadi modern, minimalis, dan premium dengan tema monochrome, dibangun menggunakan Next.js, TypeScript, dan Tailwind CSS.

## Fitur
- Desain monochrome (hitam, putih, abu-abu) yang elegan dan responsif
- Halaman utama, tentang, portofolio/proyek, dan kontak
- Navigasi premium minimalis
- Layout responsif untuk desktop & mobile

## Instalasi & Menjalankan Project

1. **Install dependencies:**
	```bash
	npm install
	```

2. **Jalankan development server:**
	```bash
	npm run dev
	```

3. Buka [http://localhost:3000](http://localhost:3000) di browser.

4. **Build untuk produksi:**
	```bash
	npm run build
	npm start
	```

## Struktur Halaman
- `/` : Beranda
- `/about` : Tentang
- `/portfolio` : Portofolio
- `/contact` : Kontak

---
Ganti `[Nama Anda]` di halaman utama dan footer dengan nama Anda sendiri.
